Zeichen und Zahlen

Bit, Byte

Ein Bit ist die kleinste Informationseinheit und kann die Werte 0 oder 1 annehmen.
Ein Byte besteht aus 8 Bit und kann somit 256 verschiedene Zustände annehmen.

Bn = {0, 1}n

ASCII-Tabelle

ASCII ist ein Zeichensatz, der 128 Zeichen umfasst und jedem Zeichen eine Zahl zuordnet.
Ein Zeichen entspricht einem Byte. (8 Bit)

Ablesen in folgender Reihenfolge:
c6 c5 c4 c3 c2 c1 c0 c-1

Festkommadarstellung

Um negative Zahlen in Binär zu repräsentieren, bestimmen wir das erste Bit und nutzen entweder die Einer-Komplement-Darstellung oder die Zweier-Komplement-Darstellung.

Einer-Komplement-Darstellung

Bei der Einer-Komplement-Darstellung werden je nach Vorzeichenbit alle anderen Ziffern invertiert.
Jedoch existiert hier das Problem der Doppeldeutigkeit der 0. (000 & 111)

Beispiel

1010 ist negativ
invertiere:
1101

Zweier-Komplement-Darstellung

Um eine Zahl in die Zweier-Komplement-Darstellung zu bringen, wird die Einer-Komplement-Darstellung angewendet und 1 addiert.

Heißt: invertieren und 1 addieren

Addition

Einer-Komplement: 1101
Zweier-Komplement: 1101 + 0001 = 1110

Gleitkommadarstellung

Das Problem an der Festkommadarstellung ist, dass sie nur eine begrenzte Anzahl an Zahlen darstellen kann.

Beispiel:
1 Bit für +/-
8 Bit für eine Zahl (vor Komma)
23 Bit für die Nachkommastellen

Lösung: Gleitkommadarstellung

Eine Gleitkommazahl besteht aus einem Vorzeichenbit, einem Exponenten und einer Mantisse.

Beispiel:
1,101 * 23

Um eine Gleitkommazahl in Dezimal zu konvertieren, wird folgende Formel angewendet:

(-1)Vorzeichen * 2Exponent * 127 * (1 + Mantisse)

Boolesche Algebra

Eine Boolesche Algebra ist durch das Tupel (M, +, *, ~) definiert sobald für beliebige x,y ∈ M folgende Axiome gelten:

Axiome

Kommutativität

x + y = y + x,
x * y = y * x

Assoziativität

(x + y) + z = x + (y + z),
(x * y) * z = x * (y * z)

Absorption

x + (x * y) = x,
x * (x + y) = x

Distributivität

x * (y + z) = (x * y) + (x * z),
x + (y * z) = (x + y) * (x + z)

Auslöschung

x + (y * (~y)) = x,
x * (y + (~y)) = x

Eigenschaften

Idempotenz

x + x = x,
x * x = x

Neutralität

x + 0 = x,
x * 1 = x

Inversität

x = ~(~x)

de-Morgan

(~x) * (~y) = ~(x + y),
(~x) + (~y) = ~(x * y)

Consensus

x * y + x * ~y = x,
(x + y) * (x + ~y) = x

Folgende algebraische Strukturen sind Boolesche Algebren:

1. (B, ∨, ∧, ¬) mit B = {0, 1}

x ∨ y = max(x, y)
x ∧ y = min(x, y)
¬x = 1 - x

2. Für eine beliebige Menge M die Teilmengenalgebra (2m, ∪, ∩, ~)

Addierer

Halbaddierer

Ein Halbaddierer ist ein Schaltkreis, der zwei Bits addiert und das Ergebnis sowie den Übertrag ausgibt.
Er besteht aus einem XOR-Gatter und einem AND-Gatter.

Volladdierer

Ein Volladdierer ist ein Schaltkreis, der zwei Bits und einen Übertrag addiert und das Ergebnis sowie den Übertrag ausgibt.
Er besteht aus zwei Halbaddierern und einem OR-Gatter.
